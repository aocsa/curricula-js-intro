## Cuenta de 0 a N

Vamos a pasar en un valor, N. Usted debe escribir un programa donde los outputs  de todos los valores vayan de 0 a más incluyendo a N.

```js

// Obtener inputs desde la línea de comandos
var N = 10 // prompt("input: ")

// Su código va aquí

for (var i = 0; i <= N; i ++) {
  console.log (i);
}


```
